# Golam Mostafa Portfolio

Personal website and portfolio for Golam Mostafa - Odoo Developer & Researcher.

## Technologies Used

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Running Locally

1. Install dependencies:
   ```sh
   npm install
   ```

2. Start the development server:
   ```sh
   npm run dev
   ```
